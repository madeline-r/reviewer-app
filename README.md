# Reviewer
> Reviewer is a web application to manage reviews for books you read

<img src="https://github.com/madeline-r/reviewer-app/blob/main/home.png" alt="drawing" width="1000"/>

## Technologies
* NodeJS v12.16.1
* MongoDB
* Express
* Marked
* Slugify
* Dompurify
* Bootstrap v5.0.0

## Setup
To start the project download the zip file, 
open in your code editor (suggested - Visual Studio Code) and 
enter run servstart in the terminal.
